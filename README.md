# Site Mangue - Landing Page

Site vitrine pour Mangue, une entreprise de jus et confiture artisanale 100% mangue.

## Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5.3

## Structure du projet

```
Site Mangue/
├── index.html          # Page principale
├── css/
│   └── styles.css      # Styles personnalisés
├── public/
│   └── images/         # Images et icônes
├── node_modules/       # Dépendances Bootstrap
└── package.json        # Configuration npm
```

## Installation

1. Installer les dépendances :
```bash
npm install
```

2. Ouvrir le fichier `index.html` dans votre navigateur

## Fonctionnalités

- Navigation responsive avec Bootstrap
- Section hero avec image de fond
- Sections produits avec alternance texte/image
- Section "Pourquoi nous choisir"
- Footer avec informations de contact
- Design responsive pour mobile et desktop

## Couleurs de la marque

- Orange principal: #F15A24

- **Inter** (700, 800) - Titres et navigation
